show(options) → {MessageView}

Shows a new message. Available options are: - text (required string) - Message text to show. Can contain HTML. - time (optional number) - Number of milliseconds after which the message will disappear. Defaults to 0, which means that it will not disappear on its own - call to `hide()` or user clicking on it is required. - type (optional string) - Type of the message. Determines its look. Available values are: `success`, `warning` and `error`.
